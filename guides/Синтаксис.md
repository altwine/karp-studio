# ИСР КАРП - Синтаксис

## Структура программы
```
алг ИмяПрограммы
    тело
```

---

## Переменные
```
счёт := 0              # объявление
имя := "Вася"
счёт = 10              # присваивание
счёт += 1
счёт -= 1
счёт *= 2
счёт /= 2
```

---

## Выражения
```
Арифметика: + - * / ( )
Сравнения: < <= > >= ==  (возвращают 1 или 0)
```

---

## Перечисления
```
перечисление Цвет:
    КРАСНЫЙ
    ЗЕЛЁНЫЙ
    СИНИЙ

использование: Цвет.КРАСНЫЙ
```

---

## Условие
```
если x > 0:
    печать("положительное")
иначе:
    печать("ноль или отрицательное")

Блоки — отступами.
```

---

## Цикл
```
пока условие:
    тело
```

---

## Выбор
```
выбор значение:
    случай ВАРИАНТ1:
        печать("первый")
    случай ВАРИАНТ2:
        печать("второй")
    иначе:
        печать("другой")
```

---

## Встроенные функции
```
печать(значение, ...)

случайное_число(мин, макс)
синус(угол)          # угол в градусах
косинус(угол)
тангенс(угол)
арксинус(x)
арккосинус(x)
арктангенс(x)

случайный_элемент(перечисление)
```

---

## Черепаха
```
вперед(расстояние)
назад(расстояние)
направо(угол)
налево(угол)
домой()

поднять_перо()
опустить_перо()
толщина_пера(пиксели)
цвет_пера(r, g, b)      # 0-255
цвет_пера("hex")
цвет_фона(r, g, b)
цвет_фона("hex")
очистить()
включить_сетку()
выключить_сетку()
показать_черепаху()
спрятать_черепаху()
```

```
функция имя(параметр1: тип, параметр2: тип) -> тип_возврата:

```

## Пример
```
алг Квадрат

счёт := 0
цвет_пера(255, 0, 0)

пока счёт < 4:
    вперед(100)
    направо(90)
    счёт += 1

печать("готово")
```
